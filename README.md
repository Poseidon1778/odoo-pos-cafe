# Odoo POS Cafe

A restaurant POS system (hackathon project) built with the PERN stack (PostgreSQL, Express, React, Node.js) — no ORM, raw SQL via node-postgres.

## Features
- Table-based ordering (Floor/Table view)
- Multiple payment methods (Cash, Digital, UPI QR)
- Kitchen Display integration
- Customer Display
- POS backend configuration + reporting dashboards
- Self/Online ordering via token
- POS session management

## Tech Stack
- PostgreSQL (raw SQL, pg driver)
- Express.js
- React (Vite)
- Node.js
- JWT authentication
